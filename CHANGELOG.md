# 更新日志

本文件记录项目的所有重要变更。项目遵循语义化版本规范。

## 1.0.1

- 包名调整为 `codepilot-ai`。
- `.gitignore` 初始化支持幂等更新和主动关闭。
- `--branch` 作为创建或切换 Git 分支的实际开关。
- 统一工具文件生成目标与验证目标。
- 确保全局 CLI 参数在子命令执行前生效。
- 增加集成运行时解析，并加固备份创建和恢复流程。
- 创建变更时保留所有配置字段，增加配置 Schema v2 迁移备份。
- 增加 `CODEPILOT_*` 环境变量，兼容旧变量并支持实际的预演模式。
- 检查流程支持内置检查和 `script:<name>` 包脚本配置。
- 危险操作增加显式确认，支持按需启用可信本地插件。
- 增加官方集成资源解析、跨平台 CI、CodeQL、Dependabot、SBOM 和发布产物。
